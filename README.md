## 📝 소개
'얼마냐면???'은 메뉴 기반 맛집 가격 비교 서비스입니다. 특정 메뉴를 검색하면 해당 메뉴를 판매하는 주변 식당들의 가격을 한눈에 비교할 수 있습니다. 네이버 지도와 연동하여 위치 정보를 제공하고, 원하는 식당을 북마크하여 저장할 수 있습니다!

### 주요 기능
- 메뉴 기반 맛집 검색
- 가격순 정렬 및 필터링
- 네이버 지도 연동으로 식당 위치 확인
- 카카오 소셜 로그인
- 식당 북마크 기능
